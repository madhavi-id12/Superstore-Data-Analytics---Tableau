# Superstore,USA Analysis-Tableau
Interactive Tableau dashboard built on the Sample Superstore (60K+ records) to analyze Sales, Profit, and Orders. Implemented KPIs (Profit Margin, Return Rate), dynamic metric parameter, Top 5 products, LODs, sets, table calculations, and filter actions. Delivered actionable insights on trends, growth, and regional performance.

The objective of this project is to design an interactive and dynamic Tableau dashboard using the Sample Superstore dataset to analyze business performance across different regions, categories, and time periods.
